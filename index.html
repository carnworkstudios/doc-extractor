Analyze Panel — Quick Guide

What you're looking at

The canvas shows a structural X-ray of your PDF page. Coloured overlays are geometry layers (lines, rectangles, text positions) detected by the pipeline — not visual rendering.

H segs — horizontal rule lines V segs — vertical rule lines Rects — closed border boxes Images — raster/vector image bounds Text dots — baseline anchor points
Region types

After Re-extract, the pipeline labels content blocks. Toggle any region type in the legend to hide it from both the canvas and the next re-extraction.

Lattice — table with visible grid lines Stream — table detected by whitespace alignment Heading — section title Para — paragraph text block Box — bordered content region Header / Footer — page margins
Pipeline threshold sliders
Min confidenceHide low-confidence regions on the canvas. Does not affect extraction — visual filter only.
Y-band tolHow much vertical drift is allowed before two text items are considered different lines. Increase for skewed PDFs.
Para gapMinimum vertical gap to split two paragraphs. Increase if paragraphs are merging; decrease if lines are splitting.
Col gapMinimum whitespace width to detect a column boundary. Increase for dense two-column PDFs.
Stream confConfidence threshold for accepting a stream table. Lower = more aggressive table detection; may increase false positives.

Drag a slider to see a ghost overlay on the canvas showing exactly what that threshold affects on the current page.

Col split tool

Click on the canvas to drop a vertical column split guide. The pipeline uses these to force column boundaries the auto-detector missed. Drag to reposition, double-click to remove.

Select tool

Click a region box to select it. Drag to move, drag a corner/edge handle to resize. Double-click empty canvas space to draw a new custom region.

Del / ⌫Mark region as skipped (shown strikethrough)
Workflow
  1. Load a PDF — the canvas auto-populates with geometry layers.
  2. Hit Re-extract page to run the pipeline and see region overlays.
  3. Adjust sliders and toggle regions to tune the pipeline for this page.
  4. Use Col split or Select to fix any remaining layout errors.
  5. Hit Re-extract again — changes apply immediately to the HTML output.
  6. Switch to the Extract tab to see the final HTML and export.
